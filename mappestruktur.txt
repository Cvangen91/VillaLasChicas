VillaLasChicas – oppdatert mappestruktur

/ (rot)
  ENDRINGER_OG_TILTAK.txt   # oppsummering av endringer som er gjort i prosjektet
  eslint.config.js          # ESLint-konfigurasjon
  index.html                # HTML-templaten som laster React-appen
  mappestruktur.txt         # denne oversikten
  package.json              # avhengigheter og npm-skript
  package-lock.json         # låst versjon av npm-avhengigheter
  README.md                 # prosjektinfo og brukerveiledning
  vite.config.js            # Vite-byggekonfigurasjon

  public/                   # statiske filer tilgjengelige direkte i appen
    villa-logo.svg          # logo brukt i header og på forsiden
    vite.svg                # standard Vite-ikon

  src/                      # applikasjonskildekode
    App.css                 # app-spesifikke basisstiler
    App.jsx                 # hovedkomponent som håndterer språkvalg og sender tekster videre
    index.css               # globale variabler, farger og grunnstiler
    main.jsx                # oppstart av React + BrowserRouter

    assets/                 # lokale assets i src
      react.svg             # standard React-ikon

    components/             # gjenbrukbare komponenter
      layout/
        Footer.jsx          # footer / sidebunn
        Navbar.jsx          # toppnavigasjon med språkvalg
        PageLayout.jsx      # felles sideoppsett for alle sider

      sections/             # eldre seksjonskomponenter fra tidligere oppsett
        Booking.jsx
        Calendar.jsx
        Hero.jsx
        ImageGallery.jsx
        Info.jsx

      ui/
        Button.jsx          # gjenbrukbar knappkomponent

    data/
      siteContent.jsx       # eldre innholdsfil / statisk innhold

    locales/                # oversettelser
      en.js                 # engelsk tekstinnhold
      no.js                 # norsk tekstinnhold

    pages/                  # sidefiler og side-spesifikke stiler
      Home.jsx              # startsiden
      Home.css              # styling kun for startsiden
      About.jsx             # Om villaen-siden
      About.css             # styling kun for Om villaen
      Contact.jsx           # kontaktsiden
      Contact.css           # styling kun for Kontakt
      pages.css             # felles basisstiler som deles av sidene

    routes/
      AppRoutes.jsx         # ruter for Home, About og Contact

Notater:
- `index.css` brukes til globale designvariabler og grunnstil.
- `Home.css`, `About.css` og `Contact.css` er egne filer for side-spesifikke effekter.
- `pages.css` beholdes som en liten delt base for generell side-layout.
- `components/sections/` og `data/siteContent.jsx` er eldre struktur som kan ryddes videre senere ved behov.
